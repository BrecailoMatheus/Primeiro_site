Hospedagem GitHub

Para criar um site no GitHub.com devemos ter uma conta válida e um repositório com uma arquivo index.HTML .

Hospedando um site

  1. Criar um repositório
  2. Publicar o repositório utilizando o PUBLISH
  3. Abrir o repositório no VSCode
  4. Criar o index.html
  5. Criar a versão utilizando COMMIT
  6. Enviar o repositório remoto utilizando PUSH
  7. Entrar no GitHub.com, acessar o repositório -> settings -> pages -> escolher o BRANCH do source
  8. Clicar na URL gerada

BRANCHES

São ramificações do seu projeto. O projeto prinncipal em geral fica na branch main. A branch kmain deve ter a versão final e funcional do seu projeto. O projeto pode ter branches secundários. Clique aqui para saber mais sobre branches.
